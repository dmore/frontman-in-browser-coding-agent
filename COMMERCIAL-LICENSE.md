## Overview

Frontman's server component (`apps/frontman_server/`) is available under the [AGPL-3.0](./apps/frontman_server/LICENSE) for open-source use. For organizations that require different terms — including whitelabel rights, removal of AGPL obligations, or proprietary deployment — Frontman AI offers a commercial license.

## What the Commercial License Provides

### Grant of Rights

The commercial license grants the licensee:

1. **Use and Deployment** — The right to use, modify, and deploy the Frontman Server for the licensee's own product or service.
2. **AGPL Exemption** — Removal of AGPL-3.0 obligations, including the requirement to disclose source code of modifications.
3. **Whitelabel Rights** — The right to rebrand and customize the product's user-facing elements, subject to the trademark terms in the agreement.
4. **Limited Redistribution** — The right to provide the software to the licensee's end customers as part of the licensee's hosted service. This does not include the right to redistribute the source code or offer the software as a standalone product.

### What the Commercial License Does Not Provide

- The right to sublicense the source code to third parties
- The right to redistribute the software as a standalone or self-hosted product
- Exclusive rights to the Frontman software (unless separately negotiated)
- Rights to Frontman's client-side libraries (these are already available under Apache-2.0)

## Licensee Restrictions

### Non-Compete

The licensee may not use the licensed code to create, offer, or operate a standalone product that directly competes with Frontman's core offering (AI agent orchestration platform). Whitelabeling the product as part of the licensee's broader platform is permitted; extracting the code to build a rival product is not.

### No Sub-Licensing

The licensee may not grant any third party rights to the Frontman source code. End customers of the licensee interact with the product as a service; they do not receive source code or license rights.

### No Source Redistribution

The licensee may not distribute, publish, or otherwise make available the Frontman Server source code to any third party, except as required for the licensee's internal operations (e.g., to the licensee's own employees and contractors under NDA).

### AI Cloning Prohibition

The licensee may not use artificial intelligence tools to analyze the licensed code for the purpose of creating a substantially similar or competing product. This includes, but is not limited to:

- Feeding the source code into AI systems to generate functionally equivalent code
- Using AI tools to extract and replicate the architecture, algorithms, or design patterns
- Training or fine-tuning AI models on the licensed code

This restriction survives termination of the agreement for twenty-four (24) months.

## Licensee Obligations

### Attribution

Unless otherwise agreed in writing, the licensee must include a "Powered by Frontman" notice in the product's user interface or documentation. Specific placement and format are defined in the executed agreement.

### Security Notification

The licensee must promptly notify Frontman AI of any security vulnerabilities discovered in the licensed code. Frontman AI will coordinate responsible disclosure and remediation.

### Audit Rights

Frontman AI may audit the licensee's use of the licensed code up to once per year to verify compliance with the license terms. Audits will be conducted with reasonable notice and during normal business hours.

### Upstream Contributions

The licensee is encouraged (but not required) to contribute non-proprietary improvements to the Frontman open-source project. Contributions are subject to the [Contributor License Agreement](./CLA.md).
