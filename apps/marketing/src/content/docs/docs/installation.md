title: Installation
description: How to install Frontman in your project.
---

Frontman ships as a framework-specific integration — pick the guide that matches your project.

## Astro

```bash
npx astro add @frontman-ai/astro
```

Then follow the [Astro integration guide →](/docs/integrations/astro/)

## Next.js

```bash
npx @frontman-ai/nextjs init
```

Then follow the [Next.js integration guide →](/docs/integrations/nextjs/)

## Vite

```bash
npm install @frontman-ai/vite
```

Then follow the [Vite integration guide →](/docs/integrations/vite/)

## WordPress (Beta)

WordPress uses a standalone binary — no npm package required.

See the [WordPress setup guide →](/docs/integrations/wordpress/)
